Cloud Computing Project


The Cloud Computing module project requires you to build a web application and have it hosted on at least two cloud providers.

The project is worth 40% of your module. Requirements • Web Application must be developed using PHP. • Application should consume at least two services, getting some information and do something useful with the information retrieved. • Application should be deployed on two cloud platforms, if you require access to Amazon, contact Rob Duncan at rduncan@ncirl.ie for an access key. Marking Scheme 40% of the marks awarded for the successful deployment of your application on two different cloud platforms. 50% of the marks awarded for the application developed in PHP. 10% of the marks awarded for a technical document (Template will be provided on Moodle).
